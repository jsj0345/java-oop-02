# Java OOP 02

Java 객체지향 프로그래밍의 기본 개념을 학습하고 예제 코드로 정리한 저장소입니다.

클래스와 객체, 참조값, 생성자, 패키지, 접근 제어자, static, final, 상속, 다형성 등 Java 객체지향 학습에 필요한 핵심 개념을 예제 코드로 정리했습니다.

## 학습 목적

Java 백엔드 개발에서 자주 사용되는 객체지향 개념을 이해하기 위해 정리했습니다.

단순히 문법을 학습하는 것보다, 클래스가 객체로 생성되는 흐름과 객체 간 관계를 이해하고, 상속과 다형성을 통해 코드 확장성과 재사용성을 높이는 방식을 익히는 데 중점을 두었습니다.

## 학습 내용

- 클래스와 객체 생성
- 필드와 메서드 사용
- 참조값과 객체 전달 방식
- 생성자를 활용한 객체 초기화
- 패키지를 통한 클래스 분리
- 접근 제어자를 활용한 캡슐화
- static 변수와 static 메서드
- final 키워드 사용
- 상속을 통한 기능 확장
- 메서드 오버라이딩
- 다형성을 활용한 객체 처리
- JVM 메모리 구조 기초

## 학습 포인트

- 클래스와 객체의 차이를 이해하고 객체 생성 흐름을 학습했습니다.
- 생성자를 사용해 객체 초기화 책임을 명확히 분리했습니다.
- 접근 제어자를 통해 외부에서 직접 접근하면 안 되는 데이터를 보호하는 방식을 익혔습니다.
- static과 final 키워드를 통해 클래스 단위 공유 자원과 변경 불가능한 값을 구분했습니다.
- 상속과 오버라이딩을 통해 기존 기능을 확장하는 방법을 학습했습니다.
- 다형성을 활용해 부모 타입으로 여러 자식 객체를 처리하는 구조를 연습했습니다.
